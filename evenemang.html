Ett event för alla

Ett evenemang tillägnat allas integration, oavsett land, kultur du kommer ifrån, religion eller din tro.
Detta evenemang var utformat för att föra människor samman för att lära sig om andras kulturer och komma närmare och känna empati med människor från hela världen.

Ett helt gratis event där vi hoppas att alla kan delta, det blir musik, mat och många överraskningar
